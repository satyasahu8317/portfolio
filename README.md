# Personal Portfolio Website

A modern, responsive personal portfolio website for a B.Tech student and Machine Learning Developer.

## Features

- **Dark Theme**: Modern dark UI with smooth animations
- **Responsive Design**: Optimized for both mobile and desktop
- **Smooth Scrolling**: Seamless navigation between sections
- **Interactive Elements**: Hover effects, mobile menu, and animations
- **Sections Included**:
  - Hero Section with call-to-action
  - About Me with profile image
  - Skills with progress bars
  - Projects showcase
  - Experience details
  - Education background
  - Contact information

## Tech Stack

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla)
- Font Awesome for icons

## Project Structure

```
portfolio/
├── index.html          # Main HTML file
├── assets/
│   ├── css/
│   │   └── style.css   # Custom CSS styles
│   └── js/
│       └── script.js   # JavaScript functionality
└── README.md           # This file
```

## Setup Instructions

1. Clone or download the project files
2. Open `index.html` in your web browser
3. The website is ready to use!

## Customization

### Personal Information
- Replace `[Your Name]` with your actual name throughout the HTML file
- Update the profile image in the About section
- Modify the skills, projects, experience, and education sections with your information
- Update contact links (email, LinkedIn, GitHub)

### Styling
- The website uses Tailwind CSS via CDN, so no build process is required
- Custom styles are in `assets/css/style.css`
- Colors can be adjusted by changing the Tailwind classes

### Adding Projects
- Add more project cards in the Projects section
- Follow the existing structure for consistency

## Browser Support

- Mobile browsers (iOS Safari, Chrome Mobile)
